Add a Material shell to your Maera theme.


== Description ==

**Build beautiful, bold sites that make an impression based on the materialize CSS framework!**

Materialize is a CSS framework that was built according to Google’s Material Design specs.

Featuring a wide selection of color combinations and unique features you’ll be able to bring your site back to life.

**Powerful and simple customizer**

With a powerful yet simple customizer you can make real decisions about your site without being lost in a sea of options and settings, focusing on what really matters.

**Colors**
You can select custom colors on a per-page basis, and also per-widget!
